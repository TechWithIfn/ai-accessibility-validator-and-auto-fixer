# Extension Icons

This directory should contain the extension icons.

## Required Icons

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Creating Icons

1. **Design your icon:**
   - Use a simple, recognizable design
   - Ensure it's readable at small sizes (16x16)
   - Use high contrast colors

2. **Generate icons:**
   - Create a 128x128 version first
   - Scale down to 48x48 and 16x16
   - Use image editing software or online tools

3. **Online Tools:**
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/
   - https://favicon.io/

4. **Quick Solution:**
   - Use any 16x16, 48x48, and 128x128 PNG images
   - Name them: `icon16.png`, `icon48.png`, `icon128.png`
   - Place them in this `icons` folder

## Example Icon

A simple accessibility symbol (checkmark, shield, or eye) on a blue background works well.

## Temporary Icons

Until you create custom icons, you can use placeholder images or simple colored squares with text.
